# 사자성어 한 수 - Layout Structure

## 세계관: 묵향서재 (墨香書齋)
전통 서재에서 매일 하나의 두루마리를 펼쳐 사자성어를 풀고,
정답 시 인장(도장)을 찍어 수집하는 수련의 경험.

## UI 설계 원칙 적용
- **주제 매칭**: 두루마리(scroll-card), 인장(stamp-mark), 먹물(ink-well), 붓(brush-divider)
- **비정형 배치**: 한자 히어로가 뷰포트 지배, 보조 정보는 작게
- **정보 덩어리화**: 뜻+예문+유래를 하나의 두루마리(result-scroll)로 통합
- **점수/등급 금지**: 먹물 농도(진묵/농묵/중묵/담묵/연묵/백지) 메타포
- **← 뒤로가기 제거**: 토스 네비바 사용 (history.pushState + popstate)
- **AD 사전 고지**: 모든 AD 버튼에 배지 + "광고 시청 후 이용 가능" 문구

## Screen Flow
```
home → reveal (1.8s auto) → result
                              ├→ collection (AD, pushState) → 토스 back → result
                              └→ stats (AD, pushState) → 토스 back → result

Streak Shield modal: shown on app open if yesterday missed + streak > 0
```

## 차별화 체크리스트

| 문제 형식 | 텍스트 4지선다 | 텍스트 4지선다 | **한자 빈칸 채우기** | O |
